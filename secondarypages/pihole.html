🧠 Building a Network-wide Ad Blocker with Pi-hole

Date: October 2025

This project demonstrates how I deployed Pi-hole on a Raspberry Pi to block ads, trackers, and malicious domains across my entire network, enhancing both privacy and performance.

⚙️ System Setup

  • Device: Raspberry Pi Zero 2 WH
  • OS: Raspberry Pi OS (Debian-based)
  • DNS Resolver: Pi-hole v5
  • Network: Home router (DNS pointing to Pi-hole)

🧩 Installation Steps

sudo apt update && sudo apt upgrade -y
sudo nano /etc/dhcpcd.conf   # Set static IP
curl -sSL https://install.pi-hole.net | bash

🖥️ Configuration

Access the web admin interface at: http://192.168.1.250/admin

  • Selected upstream DNS: Cloudflare (1.1.1.1)
  • Enabled ad-block lists via Gravity
  • Integrated Fail2Ban for protection
