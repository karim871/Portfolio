01. About Me

Hello! I'm AbdelKrim, a recent Computer Security & Networking graduate from Collège Bois-de-Boulogne with a passion for defending digital infrastructure.

My journey into cybersecurity started with a fascination for how systems communicate—and how they break. Today, I run a personal Mini-SOC using Suricata, Splunk, and the ELK Stack, where I simulate real-world attacks and tune detection rules myself. I managed to reduce false positives by ~40% through custom Suricata rule optimization.

I specialize in Networking and building automation tools that bridge the gap between threat intelligence and response.

When I'm not analyzing packets or writing detection logic, you can find me exploring new CVEs, contributing to open-source security tools, or hiking in the Montreal area.

Trilingual: English | French | Arabic

03. Some Things I've Built

Featured Project

Threat Intelligence Automation Bot

A Python-based automation framework that integrates AbuseIPDB, VirusTotal, and AlienVault OTX APIs. It automates the enrichment of Indicators of Compromise (IOCs), calculates threat scores, and generates automated reports for SOC analysts.
  • Python
  • REST APIs
  • JSON Parsing
  • Linux

Featured Project

Home Mini-SOC Lab

A full-scale Security Operations Center built on a Raspberry Pi. Features Suricata IDS with the Emerging Threats ruleset (30,000+ rules) feeding into Splunk SIEM for real-time detection, log correlation, and custom dashboard visualization.
  • Suricata
  • Splunk
  • Raspberry Pi
  • rsyslog
  • Linux

Featured Project

Network-Wide DNS Security Layer

Implementation of Pi-hole as a DNS sinkhole to block advertisements and malicious domains across the entire home network. Includes custom blocklists, regex filtering, and whitelist management for operational security.
  • Pi-hole
  • DNS Security
  • Network Filtering
  • Linux

Featured Project

pfSense Firewall — Secure Network Architecture

Deployed a full LAN/WAN segmented network using pfSense with VLAN isolation, site-to-site VPN, and OpenVPN client access. Configured advanced filtering policies, NAT, DNS Resolver, DHCP, and integrated Suricata in inline IDS mode for active traffic inspection and logging.
